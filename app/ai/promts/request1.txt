Задание: 
Необходимо проанализировать список товаров и вывести названия групп, на которые делятся товары из списка.
Всего должно быть от 2 до 10 названий групп товаров.
Имена групп товаров должны быть на кириллице.

Вывод оформить в виде Json файла, массива групп товаров
Пример выходных данных:
{
    "groups": [
        {
            "name": "Название группы товаров 1"
        },
        {
            "name": "Название группы товаров 2"
        },
        {
            "name": "Название группы товаров 3"
        },
        {
            "name": "Название группы товаров 4"
        }
	...
    ]
}
